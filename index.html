    Slices # Go slices are small constructs backed by a traditional array. The slice construct is light weight because it contains only a pointer to it’s first element, a length and a capacity. The slice pointer can point to any element in the backing array. The length denotes the number of elements currently in the slice and the capactiy denotes the maximum number of elements allowed in the slice.
